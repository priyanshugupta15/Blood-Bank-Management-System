# Contributing to Blood Bank Management System (BBMS)

Thank you for your interest in this project! This repository is maintained by a single contributor.

---

## 📖 For Learners & Developers

This project is open-source and available for educational purposes. You are welcome to:

- **Study** the codebase to learn MERN stack development
- **Fork** this repository for your own learning or customization
- **Report issues** if you find any bugs
- **Suggest features** by opening an issue

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone <your-fork-url>
cd blood-bank-management-system
```

### Install Dependencies

**Backend:**
```bash
cd backend
npm install
```

**Frontend:**
```bash
cd ../frontend
npm install
```

### Environment Setup

Create a `.env` file inside the `backend` folder:

```bash
MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

### Run the Project

**Backend:**
```bash
cd backend
npm start
```

**Frontend:**
```bash
cd frontend
npm run dev
```

## 📝 Development Guidelines

If you fork this project and make improvements, consider:

- Writing clean, readable code
- Adding comments for complex logic
- Using meaningful commit messages
- Testing your changes before committing
- Not committing `.env` or sensitive data

## 🐛 Reporting Issues

Found a bug? Please open an issue with:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Screenshots (if applicable)

---

## 💡 Feature Requests

Have an idea? Feel free to:
- Open an issue describing the feature
- Explain the use case and benefits
- Suggest implementation approach (optional)
